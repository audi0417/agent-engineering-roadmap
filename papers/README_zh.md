## 如何用 Agent Engineering 的角度讀論文？

閱讀論文時，建議關注工程化意義：

1. 這篇論文解決什麼系統問題？
2. 它改善的是哪個 Agent component？
3. 它影響的是 planning、tools、memory、workflow、evaluation 還是 safety？
4. 這個概念能不能變成 hands-on example？
5. 它的限制是什麼？
