# Launch Kit

Use this folder when announcing the Agent Engineering Roadmap.

## Short Description

Agent Engineering Roadmap is an open-source course for building production-aware AI agents: MCP tools, memory, workflows, multi-agent systems, evaluation, safety, and domain agent colonies.

## Launch Post

```text
I built an open-source Agent Engineering Roadmap for people moving from chatbot demos to production-aware agent systems.

It includes:
- complete course syllabus
- runnable Python examples
- guided labs
- instructor-ready lesson plans
- SVG diagrams
- showcase demos
- templates for memory, safety, MCP, and evals
- capstone project

Repo: https://github.com/audi0417/agent-engineering-roadmap
```
