# Visual Assets

This folder contains visual teaching assets for the Agent Engineering Roadmap.

Use these diagrams in README files, slides, workshops, or classroom material.

## Diagrams

| Diagram | Purpose |
|---|---|
| [Course Map](diagrams/course-map.svg) | Show the full learning path |
| [Agent Loop](diagrams/agent-loop.svg) | Explain the minimal agent loop |
| [Tool And MCP Layer](diagrams/tool-mcp-layer.svg) | Explain tool calling and MCP integration |
| [Memory System](diagrams/memory-system.svg) | Explain memory write, retrieve, audit, and delete |
| [Workflow Orchestration](diagrams/workflow-orchestration.svg) | Explain planner, executor, reviewer, and retry |
| [Agent Colony](diagrams/agent-colony.svg) | Explain supervisor and specialist agents |
| [Production Stack](diagrams/production-stack.svg) | Explain production evaluation and safety layers |

## Design Notes

These diagrams are intentionally simple. They are meant to help learners build a mental model before they read implementation details.
