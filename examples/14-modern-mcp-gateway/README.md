# Example 14: Modern MCP Gateway

This example teaches a modern MCP-style gateway shape.

It is not a full MCP implementation. It demonstrates the product-facing concepts learners need:

- tools
- resources
- prompts
- authorization
- elicitation when required arguments are missing

## Run

```bash
python examples/14-modern-mcp-gateway/main.py
```

## Learning Check

- Which token can call tools?
- Which operation requires elicitation?
- Why should resources and tools have separate permissions?
