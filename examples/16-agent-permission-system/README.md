# Example 16: Agent Permission System

This example treats agents as identities with owners, scopes, risk tiers, and audit logs.

## Run

```bash
python examples/16-agent-permission-system/main.py
```

## What It Shows

- agent identity
- scoped permissions
- least privilege
- permission denial
- access review
- audit log

## Learning Check

- Which agent owns which scopes?
- Which actions should require approval?
- How often should agent access be reviewed?
