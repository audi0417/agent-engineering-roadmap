# 範例 16：Agent Permission System

這個範例把 Agent 當成有 owner、scope、risk tier 與 audit log 的身份。

## 執行

```bash
python examples/16-agent-permission-system/main.py
```

## 展示重點

- agent identity
- scoped permissions
- least privilege
- permission denial
- access review
- audit log

## 學習檢查

- 哪個 Agent 擁有哪些 scopes？
- 哪些 action 應該需要 approval？
- Agent access 多久應該 review 一次？
