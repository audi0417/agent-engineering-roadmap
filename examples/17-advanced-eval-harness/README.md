# Example 17: Advanced Eval Harness

This example teaches release-oriented agent evaluation.

## Run

```bash
python examples/17-advanced-eval-harness/main.py
```

## What It Shows

- regression cases
- safety cases
- adversarial cases
- golden trace checks
- release gate pass/fail
- score breakdown by case type

## Learning Check

- Which cases should block a release?
- Which trace steps prove that a guardrail fired?
- What new eval should be added after every incident?
