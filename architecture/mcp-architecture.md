# MCP Architecture

## Overview

Model Context Protocol (MCP) provides a standard interface for connecting AI applications to external tools, resources, and data sources.

## Core Concepts

- MCP Client
- MCP Server
- Tools
- Resources
- Prompts
- Transport
- Permission boundaries

## Example MCP Servers

- Filesystem MCP
- Database MCP
- Browser/Search MCP
- Memory MCP
- Healthcare Record MCP

## Design Principle

MCP should be treated as the integration layer of an agentic system, not just a plugin mechanism.
