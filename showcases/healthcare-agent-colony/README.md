# Showcase - Healthcare Agent Colony

This demo routes health-related requests through a safety-first agent colony.

It does not provide diagnosis, treatment, or medical advice. It demonstrates routing, education boundaries, escalation, and safe output formatting.

## Run

```bash
python main.py
```

## What This Demonstrates

- supervisor routing
- safety classification
- education vs medical advice boundary
- escalation rules
- final evaluator check
