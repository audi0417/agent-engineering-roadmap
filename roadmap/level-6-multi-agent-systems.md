# Level 6 — Multi-Agent Systems

## Goal

Build systems where multiple agents collaborate, critique, debate, or specialize.

## Topics

- Supervisor pattern
- Worker agents
- Debate pattern
- Reflection pattern
- Role specialization
- Agent communication
- Conflict resolution

## Hands-on Projects

- Researcher + Writer + Reviewer system
- Coder + Tester + Reviewer system
- Healthcare specialist agent team
- Finance research agent team

## Outcome

You should understand when multi-agent systems add value and when they introduce unnecessary complexity.
