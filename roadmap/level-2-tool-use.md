# Level 2 — Tool Use

## Goal

Teach agents how to interact with external tools and APIs.

## Topics

- Function calling
- Tool schemas
- Tool selection
- Tool chaining
- Tool result validation
- Safe tool execution

## Hands-on Projects

- Calculator tool agent
- File reader agent
- Database query agent
- Browser/search agent

## Outcome

You should be able to design tools that agents can call reliably and safely.
