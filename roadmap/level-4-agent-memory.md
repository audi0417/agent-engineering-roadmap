# Level 4 — Agent Memory

## Goal

Design memory systems that allow agents to retain useful context across tasks and sessions.

## Topics

- Short-term memory
- Episodic memory
- Semantic memory
- User memory
- Shared memory
- Memory write policies
- Memory retrieval and ranking
- Memory safety

## Hands-on Projects

- Personal memory agent
- Long-term task memory
- Shared memory for multiple agents
- Healthcare memory prototype

## Outcome

You should be able to build agents that remember useful information without polluting context or leaking sensitive data.
