# Level 3 — Model Context Protocol (MCP)

## Goal

Use MCP to standardize how agents connect to tools, data sources, and external systems.

## Topics

- MCP client
- MCP server
- MCP tools
- MCP resources
- MCP prompts
- Local and remote MCP servers
- Security boundaries

## Hands-on Projects

- Filesystem MCP server
- Database MCP server
- Memory MCP server
- Health record demo MCP server

## Outcome

You should understand how MCP can become the integration layer for agentic applications.
