# Level 0 — AI & LLM Fundamentals

## Goal

Understand the foundations required to build AI agent systems.

## Topics

- Large Language Models
- Prompt engineering
- Context windows
- Embeddings
- Vector databases
- Retrieval-Augmented Generation
- Structured outputs
- Evaluation basics

## Hands-on Projects

- Build a basic LLM chat interface
- Create an embedding search demo
- Build a simple RAG assistant

## Outcome

You should understand how LLM applications work before adding tools, memory, and multi-agent orchestration.
