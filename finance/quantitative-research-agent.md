# Quantitative Research Agent

## Goal

Design an agent that assists with quantitative research workflows, including data collection, factor analysis, backtesting, and experiment reporting.

## Capabilities

- Retrieve market data
- Generate factor candidates
- Run backtests
- Compare strategies
- Summarize experiment results
- Track research memory

## Example Use Cases

- Multi-factor stock screening
- Reinforcement learning trading experiment review
- GNN-based signal research
- Risk-controlled portfolio construction
