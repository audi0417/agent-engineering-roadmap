# Finance Agent Colony

## Goal

Build a finance-oriented agent colony for market research, factor analysis, portfolio construction, and risk monitoring.

## Example Agents

- Market Research Agent
- Factor Analysis Agent
- Portfolio Agent
- Risk Agent
- Backtesting Agent
- Report Agent

## Example Flow

```text
Market Question
  ↓
Research Agent
  ↓
Factor Agent + Data Tools
  ↓
Portfolio Agent
  ↓
Risk Agent
  ↓
Investment Research Report
```

## Safety Notes

Finance agents should support research and analysis. They should not be treated as personalized investment advice.
