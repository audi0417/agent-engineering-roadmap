# Taiwan Health MCP

## Goal

Design a healthcare MCP layer for Taiwan-oriented health data, care workflows, and AI agent integration.

## Possible MCP Tools

- Read patient profile
- Read nutrition records
- Read vital signs
- Read care notes
- Create follow-up task
- Generate care summary

## Possible Resources

- Health records
- Nutrition logs
- Blood pressure records
- Blood glucose records
- Care management notes

## Future Direction

This track can become a reference implementation for healthcare agent systems in Taiwan.
