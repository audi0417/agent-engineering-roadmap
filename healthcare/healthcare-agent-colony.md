# Healthcare Agent Colony

## Goal

Build a healthcare-oriented agent colony for care management, nutrition tracking, health monitoring, and follow-up workflows.

## Example Agents

- Care Manager Agent
- Nutrition Agent
- Vital Sign Agent
- Psychology Agent
- Medication Agent
- Memory Agent
- Safety Evaluator Agent

## Example Flow

```text
Patient Data
  ↓
Care Manager Agent
  ↓
Specialist Agents
  ↓
Taiwan Health MCP
  ↓
Shared Health Memory
  ↓
Follow-up Recommendation
```

## Safety Notes

Healthcare agents should provide decision support, not autonomous medical diagnosis or treatment.
