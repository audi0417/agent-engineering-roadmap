# Lab 10 - Capstone Prep

## Objective

Prepare a complete agent system design before implementation.

## Estimated Time

90 minutes

## Read First

- `projects/capstone-agent-colony.md`
- `patterns/README.md`
- `templates/agent-spec-template.md`

## Task

Design your capstone agent system.

Your design must include:

- target user
- task scope
- agents
- tools
- memory
- workflow graph
- evaluation suite
- safety gates
- deployment plan

## Acceptance Criteria

Your design is ready when another engineer can implement the first version without asking what the system is supposed to do.

## Extension

Create a release checklist for version 0.1.
