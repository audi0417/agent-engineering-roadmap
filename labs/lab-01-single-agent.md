# Lab 01 - Single Agent

## Objective

Build a single-purpose agent with a clear role and structured output.

## Estimated Time

45-60 minutes

## Read First

- `curriculum/01-agent-architecture.md`
- `examples/01-single-agent/README.md`

## Task

Modify Example 01 into a new single-purpose agent.

Choose one:

- extract action items from meeting notes
- classify support tickets
- create a study plan from a goal
- summarize a research article

## Acceptance Criteria

Your agent must:

- have a narrow role
- include at least three rules
- produce predictable Markdown or JSON sections
- avoid inventing facts
- handle empty or unclear input

## Extension

Add a small test file with five inputs and expected behaviors.
