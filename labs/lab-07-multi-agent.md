# Lab 07 - Multi-Agent Systems

## Objective

Coordinate specialized agents without creating chaos.

## Estimated Time

60-90 minutes

## Read First

- `curriculum/07-multi-agent-systems.md`
- `examples/06-agent-colony/README.md`

## Task

Add one new domain agent to Example 06.

Possible agents:

- education agent
- legal intake agent
- customer support agent
- data analysis agent

## Acceptance Criteria

Your new agent must include:

- routing keywords
- disclaimer or boundary
- recommended workflow
- human review rule
- evaluation requirement

## Common Mistake

More agents can make the system worse if no one owns the final answer.

## Extension

Add a conflict-resolution policy when two agents both seem relevant.
